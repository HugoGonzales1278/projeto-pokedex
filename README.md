# 📖 Projeto Pokédex

Uma Pokédex interativa construída com HTML, CSS e JavaScript puro, que consome a [PokéAPI](https://pokeapi.co/) para exibir informações detalhadas dos Pokémon em cards estilizados.

![Licença MIT](https://img.shields.io/badge/license-MIT-green)
![HTML](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

---

## ✨ Funcionalidades

- 🔍 Busca e exibição de Pokémon via PokéAPI
- 🃏 Cards estilizados com informações de cada Pokémon
- 📱 Interface responsiva
- ⚡ Sem dependências externas — HTML, CSS e JS puro

## 🛠️ Como Rodar Localmente

1. **Clone o repositório:**

```bash
git clone https://github.com/HugoGonzales1278/projeto-pokedex.git
```

2. **Acesse a pasta:**

```bash
cd projeto-pokedex
```

3. **Abra o arquivo `index.html` no navegador.**

> Não é necessário instalar nenhuma dependência!
